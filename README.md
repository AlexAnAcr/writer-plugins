# Создатель плагинов для 

Данная программа позволяет удобно создавать плагины для . Благодаря чему  может быть удобно дополнено нужным функционалом.

Если у вас на примете есть человек, который хочет стать программистом, а вы не хотите, чтобы он стал программистом, посоветуйте ему ide WriterPlugins.
